Cara Setup Google Sheets
1. Buka Google Sheets → Extensions → Apps Script
2. Paste isi file Code.gs, ganti SECRET_KEY → jalankan setupSheets()
3. Deploy → New Deployment → Web App
Execute as: Me | Access: Anyone → Copy URL
4. Paste URL + Key di atas → Klik Hubungkan ✅
